Dựa vào ảnh bạn gửi, mình thấy repository của bạn có file TrangChu.HTML ở thư mục gốc, nhưng GitHub Pages vẫn không hiển thị đúng trang này. Bạn có thể thử các bước sau: --- 1. Truy cập trực tiếp đường dẫn Truy cập thử: https://mrpool3107.github.io/Tousehao-app/TrangChu.HTML Nếu trang không hiển thị đúng: Kiểm tra lại xem tên file có đúng TrangChu.HTML không (chú ý viết hoa). Nếu bạn muốn trang chủ tự động mở file này, cần tạo thêm index.html. --- 2. Tạo file index.html để chuyển hướng (Quan trọng) Do GitHub Pages mặc định tìm index.html, bạn cần tạo file này với nội dung sau:

Nếu trình duyệt không tự động chuyển hướng, bấm vào đây.

File này sẽ tự động chuyển hướng từ index.html sang TrangChu.HTML. Sau khi tạo file, commit & push lên GitHub. --- 3. Kiểm tra cài đặt GitHub Pages Vào Settings > Pages. Kiểm tra lại Source (nhánh main, thư mục /root). Nếu có thay đổi, nhấn Save và đợi vài phút để cập nhật. --- 4. Xóa cache trình duyệt & thử lại Nếu bạn đã cập nhật nhưng vẫn không thấy thay đổi, thử: Xóa cache trình duyệt bằng Ctrl + Shift + Delete (Chrome). Hoặc mở link trong chế độ ẩn danh. Sau khi làm các bước trên, thử truy cập lại đường dẫn chính: https://mrpool3107.github.io/Tousehao-app/ Nếu vẫn lỗi, báo lại mình để hỗ trợ thêm nhé!
